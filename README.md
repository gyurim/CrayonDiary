**사용 기술**
- Kotlin
- RoomDB

**기능**
- Save & Modify & Delete diary
- Drawing Canvas
- Share
- Save in External Storage(ex. gallery)

**UI**
- Recycler View를 이용한 격자무늬와 목록 출력
